# Multi Crypto Price Tracker

A Python project that fetches real-time cryptocurrency prices using the CoinGecko REST API.

## Features
- Fetch Bitcoin price
- Fetch Ethereum price
- Fetch Solana price
- Fetch Binance Coin price
- HTTP request handling
- Timeout handling
- Safe JSON extraction

## Technologies
- Python
- Requests
- REST API

## Run Project

pip install -r requirements.txt
python main.py
