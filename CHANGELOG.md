# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-10

### Added

- Integrated CoinGecko API to fetch cryptocurrency prices.
- Stored cryptocurrency prices in PostgreSQL.
- Generated formatted cryptocurrency reports.
- Implemented centralized logging.
- Added retry mechanism with exponential backoff.
- Added email report delivery with attachments.
- Added Windows Task Scheduler support.
- Improved configuration management using environment variables.
- Added unit tests using pytest and unittest.mock.
